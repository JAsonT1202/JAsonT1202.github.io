🛰️ arXiv API による時系列予測論文収集

まずは、arXiv API を活用して2025年投稿の時系列予測関連論文を最大500件取得し、PDFとメタデータを保存するスクリプトです。

📁 保存形式

  • PDFファイル → pdf/2505.xxxxx.pdf
  • メタ情報 → meta_arxiv_2025_timeseries.json
