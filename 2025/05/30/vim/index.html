vimの使用方法

筆者はWHIの夏インターンを参加した時、vimの利用方法が知らなかったし、気まずいでした。ですから、本記事では、vim の命令を紹介します。

📌 本記事の内容

  • vimの使用方法

vimとは

vimはViの改良版であり、テキストエディタとして多くのプログラマーに愛用されています。軽量かつ高速で、キーボードだけでほぼすべての操作が可能です。

1. vimの起動と終了

起動方法

  • 例: vim example.txt 既存のファイルを開く/新規作成

終了方法

  • ノーマルモードで以下を入力:
    • :q - 変更を保存せずに終了
    • :q! - 強制終了（変更を破棄）
    • :wq または :x - 保存して終了

2. モードの概要

vimには主に3つのモードがあります。

  • ノーマルモード: 起動直後のモード。コマンド入力が可能。
  • 挿入モード: テキストの挿入が可能。
  • ビジュアルモード: 範囲選択が可能。

モード切替

  • 挿入モードへ: i（カーソル位置から挿入）、a（カーソルの次の位置から挿入）、o（下の行に新規行を開く）
  • ノーマルモードへ: Esc
  • ビジュアルモードへ: v（文字単位）、V（行単位）、Ctrl+v（矩形選択）

3. 基本的な移動コマンド

  • h - 左に移動
  • j - 下に移動
  • k - 上に移動
  • l - 右に移動
  • 数字を前につけると回数指定可能: 5j - 5行下へ
  • gg - ファイル先頭へ
  • G - ファイル末尾へ
  • 0 - 行の先頭へ
  • $ - 行の末尾へ

4. テキスト編集

挿入

  • i - カーソル位置から挿入
  • I - 行頭から挿入
  • a - カーソルの次の位置から挿入
  • A - 行末から挿入
  • o - 下の行に新行を作成し挿入
  • O - 上の行に新行を作成し挿入

削除

  • x - カーソル位置の文字を削除
  • dw - カーソル位置から単語の末尾まで削除
  • d$ - カーソル位置から行末まで削除
  • dd - 行全体を削除
  • D - カーソル位置から行末まで削除

コピー・貼り付け（ヤンク・プット）

  • yy または Y - 行全体をコピー
  • yw - 単語をコピー
  • y$ - カーソル位置から行末までコピー
  • p - コピーした内容をカーソルの後に貼り付け
  • P - コピーした内容をカーソルの前に貼り付け

置換

  • r<char> - カーソル位置の文字を <char> に置き換え
  • R - 上書きモード（上書きしながらテキスト入力）

5. 検索と置換

  • /パターン + Enter - 前方検索
  • ?パターン + Enter - 後方検索
  • n - 次の検索結果へ
  • N - 前の検索結果へ

置換コマンド

  • ファイル全体で検索文字列を置換
  • 行番号を指定して部分置換: :10,20s/foo/bar/g

6. 設定とカスタマイズ

  • 設定ファイル: ~/.vimrc
  • 主な設定例

7. まとめ

vimは最初こそ使い方に戸惑いますが、慣れると非常に高速かつ効率的にテキスト編集ができます。簡単に言えば、o –> 編集 –> Esc –> :wq または :qで編集できます。本記事で紹介したコマンドを手元で実践し、まずは基本操作を体得しましょう。
