論文収集とデータセット生成 ～arXivからInstruction形式まで～

本記事では、arXiv から自動で時系列予測論文を収集する Python スクリプトと、収集したPDFからInstruction形式のデータセットを生成する方法をご紹介します。

📌 本記事の内容

  • arXiv API を用いた時系列予測論文のPDF＋メタデータ取得
  • PyMuPDF (fitz) を使ったPDFテキスト抽出
  • Instruction形式データセット (JSONL) の生成

🧰 使用技術

項目 ツール
言語 Python 3.9+
論文収集 arxiv, tqdm
PDF処理 pymupdf (fitz)
データ加工 json モジュール

🛰️ arXiv API による時系列予測論文収集

まずは、arXiv API を活用して2025年投稿の時系列予測関連論文を最大500件取得し、PDFとメタデータを保存するスクリプトです。

📁 PDFテキスト抽出＆Instruction形式データセット生成

次に、ダウンロードしたPDFをPyMuPDFで読み込み、Instruction形式のサンプルを生成します。出力はJSONLフォーマットです。

📁 保存形式

  • PDFファイル → pdf/2505.xxxxx.pdf
  • メタ情報 → meta_arxiv_2025_timeseries.json
  • Instruction形式データセット → finetune_dataset_en.jsonl
