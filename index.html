論文収集入門：arXiv・国際会議から時系列予測論文を自動収集

Chatbot4TS プロジェクトの第一歩として、予測モデルの事前学習・微調整に必要な専門論文を効率よく収集することが不可欠です。本記事では、arXiv や OpenReview（ICLR、NeurIPSなど） から自動で関連論文を収集する Python スクリプトの作成方法をご紹介します。

📌 本記事の内容

  • arXiv API を用いた論文PDF＋メタデータの取得
  • OpenReview によるトップ会議（ICLR 等）の論文収集
  • 論文の保存形式と後処理のヒント
